

## 获取登录配置 用户登录注册接口


**接口地址**:`/api/doctor/login/config`


**请求方式**:`GET`


**请求数据类型**:`application/x-www-form-urlencoded`


**响应数据类型**:`*/*`


**接口描述**:


**请求参数**:


**请求参数**:


暂无


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|CommonResult«FrontLoginConfigResponse»|
|401|Unauthorized||
|403|Forbidden||
|404|Not Found||


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code||integer(int64)|integer(int64)|
|data||FrontLoginConfigResponse|FrontLoginConfigResponse|
|&emsp;&emsp;logo|登录logo|string||
|&emsp;&emsp;mobileLoginLogo|移动端登录Logo|string||
|&emsp;&emsp;routinePhoneVerification|小程序—手机号授权验证类型|string||

|message||string||


**响应示例**:
```javascript
{

	"data": {
```
